# Rackula Brand Assets

Canonical brand assets for the Rackula project.

## Logo Mark

The Rackula logo mark is a coffin-tapered rack frame with a fang notch: curved shoulders, sides that narrow toward the base, and three device slots that shrink with the taper in forced perspective. A server rack, a coffin, and a bite, all in one silhouette.

### Canonical Source

The geometry lives in `src/lib/components/logo-geometry.ts`. Everything in this directory is generated from it:

```bash
npm run generate-brand-assets
```

Do not hand-edit the SVGs or rasters here; change the geometry module and regenerate. The script requires ImageMagick (`magick`) on PATH for `favicon.ico`; all other outputs need only the repo's dependencies.

### Design Specifications

The geometry specification table lives in [`docs/reference/BRAND.md`](../../docs/reference/BRAND.md) (Logo Mark section); the numbers themselves live in the geometry module. Two intentional quirks, documented in the module, do not "correct" them: the slot taper is deliberately steeper than the body taper (exaggerated perspective), and corner radii are graduated so nearer edges are rounder.

Because the slots are punched-out holes (`fill-rule="evenodd"`), every variant works on any background; the variants differ only in fill colour.

## Favicons

### ICO (Multi-resolution)

`favicon.ico` is the legacy `/favicon.ico` fallback and contains three sizes:

- 16x16 (browser tabs)
- 32x32 (browser tabs @2x)
- 48x48 (Windows taskbar)

Modern browsers use the SVG and PNG favicon links instead; larger ICO entries are stored as uncompressed BMPs and bloat the file for no benefit.

## ASCII Banner

`ascii-banner.txt` is the terminal banner: the rack-fang logo mark stacked above the wordmark in the `pagga` figlet font. Use it for CLI splash screens, MOTDs, and READMEs where you control the output.

Note: this is not the community-scripts LXC header. Those headers are auto-generated by upstream CI as `figlet -f slant "$APP"` and cannot carry custom art. See the closing notes on issue #1954.

Stored as plain monochrome text. Apply the brand purple at display time.

### Print in Dracula Purple (`#BD93F9`)

```bash
# Solid brand purple, truecolour
while IFS= read -r line; do printf '\033[38;2;189;147;249m%s\033[0m\n' "$line"; done < ascii-banner.txt

# Dracula gradient: Dracula Purple (#BD93F9) to Pink (#FF79C6), top to bottom
awk 'BEGIN{r0=189;g0=147;b0=249;r1=255;g1=121;b1=198}
{l[NR]=$0} END{n=(NR>1?NR-1:1);
for(i=1;i<=NR;i++){t=(i-1)/n;
printf "\033[38;2;%d;%d;%dm%s\033[0m\n",r0+(r1-r0)*t,g0+(g1-g0)*t,b0+(b1-b0)*t,l[i]}}' ascii-banner.txt
```

Avoid `lolcat` here: its rainbow cycle drifts off the brand purple. Use the gradient above for a brand-accurate Dracula fade.

### Regenerating

```bash
# Wordmark (pagga figlet font via toilet)
toilet -f pagga 'RACKULA'
```

The rack mark is hand-built block art approximating the logo silhouette. Edit `ascii-banner.txt` directly to adjust it.
