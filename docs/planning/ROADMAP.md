The live plan lives in GitHub, not here. This page holds only the durable framing (vision and explicit non-goals). For current work, milestones, and sequencing, use the live sources:

---

## Vision

Rackula is a lightweight, FOSS, web-based rack layout designer for homelabbers. It prioritises:

- Simplicity: do one thing well, visual rack planning
- Offline-first: works without accounts or cloud services
- Self-hostable: deployable on your own infrastructure
- Community-driven: built by homelabbers, for homelabbers

The versioning policy (CalVer for the app, SemVer for any published packages) lives in [CLAUDE.md](../../CLAUDE.md#versioning-policy) and the [CalVer decision record](../superpowers/specs/2026-05-29-versioning-policy-calver-design.md).

---

## Out of Scope

Features that will not be implemented:

- User accounts, except as part of the opt-in hosted cloud sync service
- Backend or database requirements, beyond the optional self-host persistence API and the opt-in hosted cloud sync service
- Internet Explorer support
- Native mobile apps
